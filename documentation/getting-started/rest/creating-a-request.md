## REST API 平台

当您打开 Hoppscotch 时，默认情况下会激活 REST API 平台。Hoppscotch 允许您发起 API 请求并检查响应。

您可以输入 API 端点并从下拉菜单中选择适合您需求的 HTTP 方法。配置完成后，点击 "**发送**" 按钮，您将看到服务器返回的响应。就是这么简单。
